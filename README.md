# Customer Churn Prediction

##  Project Overview

This project focuses on predicting whether a customer is likely to leave a service or not. The goal is to help businesses understand customer behavior and take steps to improve retention.


##  Features
* Performed data cleaning and preprocessing
* Handled missing values and inconsistent data
* Converted categorical data into numerical format
* Trained and evaluated machine learning models
* Compared different models to check performance


##  Technologies Used
* Python
* Pandas
* NumPy
* Matplotlib and Seaborn
* Scikit-learn


## Models Used
* Logistic Regression
* Random Forest Classifier


## Evaluation Metrics
* Accuracy Score
* Confusion Matrix
* Classification Report (Precision, Recall, F1-score)


## Dataset
* Telecom Customer Churn Dataset
* Includes customer details like tenure, services, and charges


##  How It Works
1. Loaded the dataset
2. Cleaned and prepared the data
3. Converted categorical values into numerical form
4. Split the data into training and testing sets (80-20)
5. Trained models on the dataset
6. Evaluated performance using different metrics


##  Results
* Built a working model to predict customer churn
* Compared multiple models to find better performance
